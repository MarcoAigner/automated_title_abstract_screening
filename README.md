This repository will contain the code for the master's thesis of Marco Aigner.

The repository structure is as follows:

- environment: .yml-file to setup a conda environment with all required packages in one line
- data: copies of the datasets used in the thesis in raw format and after each step of processing
- functions: functions that are re-used throught the notebooks
- chapters: ipython-notebooks that contain the code structured according to the thesis' chapters
